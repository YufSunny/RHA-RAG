Differential Calculus

Definition 3.1 (Limit). The limit of f(x) as x approaches a is L, written lim(x->a) f(x) = L, if for every epsilon > 0 there exists delta > 0 such that 0 < |x-a| < delta implies |f(x)-L| < epsilon.

Definition 3.2 (Derivative). The derivative of f at a point a is f'(a) = lim(h->0) [f(a+h) - f(a)] / h.

Theorem 3.3 (Power Rule). d/dx (x^n) = n x^(n-1) for any real n.

Theorem 3.4 (Product Rule). (fg)' = f'g + fg'.

Theorem 3.5 (Chain Rule). d/dx f(g(x)) = f'(g(x)) g'(x).

Theorem 3.6 (Mean Value Theorem). If f is continuous on [a,b] and differentiable on (a,b), then there exists c in (a,b) with f'(c) = [f(b)-f(a)]/(b-a).

Theorem 3.7 (Fundamental Theorem of Calculus). Differentiation and integration are inverse operations: the integral from a to b of f'(x) dx equals f(b) - f(a).
