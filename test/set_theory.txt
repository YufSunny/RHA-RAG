Basic Set Theory
===============

Definition 1.1 (Set). A set is a well-defined collection of distinct objects,
called elements or members. We write a in S to mean "a is an element of S."

Definition 1.2 (Subset). A set A is a subset of B, written A subset B, if every
element of A is also an element of B.

Definition 1.3 (Union). The union of sets A and B is A U B = {x | x in A or x in B}.

Definition 1.4 (Intersection). The intersection is A n B = {x | x in A and x in B}.

Definition 1.5 (Function). A function f: A -> B assigns each element of A to
exactly one element of B.

Definition 1.6 (Injective). A function f: A -> B is injective (one-to-one) if
f(x) = f(y) implies x = y for all x,y in A.

Definition 1.7 (Surjective). A function f: A -> B is surjective (onto) if for
every b in B there exists a in A such that f(a) = b.

Definition 1.8 (Bijective). A function is bijective if it is both injective and
surjective.
