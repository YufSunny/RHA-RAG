# Introduction to Point-Set Topology

## Basic Definitions

A **topological space** is a pair (X, τ) where X is a set and τ is a collection of subsets of X satisfying:

1. The empty set ∅ and X itself belong to τ.
2. The union of any collection of sets in τ is also in τ.
3. The intersection of any finite collection of sets in τ is also in τ.

The collection τ is called a **topology** on X, and its members are called **open sets**.

## Examples of Topological Spaces

### Discrete Topology

The **discrete topology** on a set X is the topology where every subset of X is open. That is, τ = P(X), the power set of X.

### Indiscrete Topology

The **indiscrete topology** (or trivial topology) on a set X is τ = {∅, X}.

### Standard Topology on ℝ

The **standard topology** on the real line ℝ is generated by open intervals (a, b) = {x ∈ ℝ : a < x < b}.

## Continuity

A function f : X → Y between topological spaces is **continuous** if for every open set V in Y, the preimage f⁻¹(V) is open in X.

**Theorem 1.** A function f : ℝ → ℝ is continuous in the ε-δ sense if and only if it is continuous when ℝ is equipped with the standard topology.

## Compactness

A topological space X is **compact** if every open cover of X has a finite subcover.

**Theorem 2 (Heine-Borel).** A subset of ℝⁿ is compact if and only if it is closed and bounded.

## Connectedness

A space X is **connected** if it cannot be partitioned into two disjoint non-empty open sets.

The interval [0, 1] is connected, while {0, 1} with the discrete topology is not.
