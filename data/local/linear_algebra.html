Linear Algebra Fundamentals

Vector Spaces

A vector space over a field F is a set V equipped with two operations: vector addition and scalar multiplication, satisfying the following axioms:

  • Associativity of addition: u + (v + w) = (u + v) + w
  • Commutativity of addition: u + v = v + u
  • Identity element: There exists 0 ∈ V such that v + 0 = v
  • Inverse elements: For every v ∈ V, there exists −v ∈ V
  • Distributivity of scalar multiplication over vector addition
  • Distributivity of scalar multiplication over field addition

Linear Independence

A set of vectors {v₁, v₂, ..., vₙ} is linearly independent if the only solution to a₁v₁ + a₂v₂ + ... + aₙvₙ = 0 is a₁ = a₂ = ... = aₙ = 0.

Theorem (Basis Representation). Every vector in a finite-dimensional vector space can be written uniquely as a linear combination of basis vectors.

Eigenvalues and Eigenvectors

For a square matrix A, a non-zero vector v is an eigenvector with eigenvalue λ if:

Av = λv

Theorem (Spectral Theorem). A real symmetric matrix has real eigenvalues and an orthonormal basis of eigenvectors.

Inner Product Spaces

An inner product on a vector space V is a function ⟨·, ·⟩ : V × V → F satisfying:

  • Conjugate symmetry: ⟨u, v⟩ = ⟨v, u⟩
  • Linearity in the first argument
  • Positive-definiteness: ⟨v, v⟩ ≥ 0, with equality iff v = 0

The Cauchy-Schwarz inequality states that |⟨u, v⟩|² ≤ ⟨u, u⟩⟨v, v⟩.
