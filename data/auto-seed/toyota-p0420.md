# Toyota TSB Excerpt — P0420 Catalyst Efficiency

**Source:** Toyota Technical Service Bulletin (public excerpt).
**Applies to:** 2014–2018 Toyota Camry 2.5L (2AR-FE) and similar
2.5L A25A-FKS models.
**Code:** P0420 — Catalyst System Efficiency Below Threshold (Bank 1).

## Table 3 — P0420 frequency by model year

| Model year | Reported P0420 cases (per 10k vehicles) |
|---|---|
| 2014 | 12 |
| 2015 | 18 |
| 2016 | 27 |
| 2017 | 41 |
| 2018 | 33 |
| 2019 | 22 |
| 2020 | 14 |

## Section 4.2 — Likely root causes (ranked)

1. **Aging catalytic converter** (most common after 80k miles).
2. **Exhaust leak upstream of Bank 1 sensor 2** — false-low reading.
3. **Faulty downstream O₂ sensor** (Bank 1, Sensor 2).
4. **Engine misfire** allowing unburned fuel into the catalyst.

## Recommended diagnostic sequence

1. Pull freeze-frame data; check short-term / long-term fuel trim at
   the time of fault.
2. Visually inspect exhaust manifold and pre-cat pipe for soot streaks.
3. Compare upstream vs downstream O₂ sensor voltage traces at idle
   and 2500 rpm — downstream should mirror upstream less aggressively
   if the catalyst is healthy.
4. If traces look normal, replace downstream O₂ sensor before condemning
   the catalyst.
